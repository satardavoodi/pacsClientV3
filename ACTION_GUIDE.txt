## What Was Fixed

**The Problem:**
- 30+ `QObject::setParent` errors
- App completely frozen during downloads
- UI unresponsive

**The Solution:**
- All UI updates now use `QTimer.singleShot()` 
- Ensures updates happen in main thread
- Worker threads don't block
- Result: No freezing, no errors

**Files Changed:**
- `PacsClient/zeta_download_manager/ui/main_widget.py`
- 6 methods updated with thread-safe wrappers
