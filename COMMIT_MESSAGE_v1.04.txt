MAJOR FEATURES
═══════════════════════════════════════════════════════════════

1. Zeta MPR - Production Ready ✅
   - Three-view orthogonal MPR (Axial, Sagittal, Coronal)
   - Synchronized crosshairs across all views
   - Input-level left-right flip correction (v1.01)
   - Proper anatomical orientation for CT and MRI
   - Reset button functionality
   - Measurement tools integration

2. UI Improvements ✅
   - MPR button now directly launches Zeta MPR
   - Removed redundant Zeta MPR from dropdown menu
   - Streamlined user experience

3. Naming Unification ✅
   - All references unified to "Zeta MPR"
   - Consistent branding throughout application
   - No more "Standard MPR" confusion

4. Experimental Oblique MPR (v1.03-dev) ⚠️
   - Simple VTK approach using SetResliceTransform()
   - Tested successfully at 15° rotation
   - Stable v1.02 backup available for rollback
